01 · Local operation
Add two raster layers cell by cell
A local operation computes a new value for each cell using only values at that same position across layers.

You have two elevation grids, Layer A and Layer B, representing measurements from two sensors. Compute Layer C = A + B for every cell. Fill in your answers in Layer C, then check.
Layer C[i,j] = Layer A[i,j] + Layer B[i,j]
Top-left: A = 3, B = 5 → C = 8. Work row by row, position by position.

03 · Focal operation
Smooth a grid with a 3×3 mean filter
A focal operation computes each output cell using a neighbourhood of surrounding cells — not just the cell itself.

Apply a 3×3 mean filter: for each cell in the output, compute the average of the 3×3 window centred on that cell. Edge cells use only their available neighbours. Fill in all output cells and check.
Output[i,j] = mean( all cells in 3×3 window centred on [i,j] )
-- edge cells: use only the neighbours that exist
Centre cell (row 2, col 2): average all 9 values. Corner cell (row 1, col 1): only 4 neighbours available — average those 4. Round to 1 decimal.
04 · Zonal statistics
Compute the mean value per zone
Zonal statistics summarise a value raster (e.g. NDVI) within zones defined by a separate raster (e.g. land cover).

The Zone grid assigns each cell to a zone (1, 2, or 3). The Value grid contains NDVI measurements. For each zone, compute the mean NDVI. Round to 1 decimal place and fill in the summary table.
Zonal mean[z] = mean( Value[i,j] where Zone[i,j] = z )
Zone 1 cells are: (0,0)=4, (0,2)=8, (1,1)=6 → mean = (4+8+6)/3. Find all cells for each zone, then average.
05 · Threshold masking
Mask out cells below a threshold
Masking sets cells that do not meet a condition to NoData — effectively removing them from analysis. This is essential for excluding water, clouds, urban areas, or low-quality pixels.

Apply a threshold: cells where value < 5 → NoData, all others keep their value. Fill in the output grid. Write ND for NoData cells, or the original value for valid cells.
if value < 5 → NoData (masked out)
else → keep original value
Scan each cell. If its value is less than 5, write ND in the output. Otherwise copy the value as-is.
06 · Weighted overlay
Combine three criteria layers with weights
Weighted overlay combines multiple criteria layers into a single suitability score. Each layer is multiplied by its weight, and the results are summed. Weights must sum to 1.0.

You have three layers: Slope, Soil quality, and Water access. Adjust the weights using the sliders, then compute the suitability score for each cell. Round to 1 decimal.
Suitability[i,j] = w1 × Slope[i,j] + w2 × Soil[i,j] + w3 × Water[i,j]
-- constraint: w1 + w2 + w3 = 1.0
For top-left cell: Slope=3, Soil=7, Water=5. Multiply each by its weight and sum. E.g. with equal weights (0.33 each): 3×0.33 + 7×0.33 + 5×0.33 = 4.95 ≈ 5.0.
07 · Change detection
Detect vegetation change between two dates
Change detection subtracts a before-raster from an after-raster to identify where values increased, decreased, or stayed the same.

Grid T1 shows NDVI in 2010, Grid T2 shows NDVI in 2020. Compute the difference (T2 − T1), then classify each cell: Gain if difference > 1, Loss if difference < −1, Stable otherwise. Fill in the difference values and the change class.
