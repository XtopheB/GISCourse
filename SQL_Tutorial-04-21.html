4a · Land productivity — yield (ProdKg_mod / Areaha_mod)
Calculate yields (defined as Production / surface) in the table Agriculture
Hint: Areaha_mod (surface) has to be positive!
Divide two columns with /: ProdKg_mod / Areaha_mod AS yield. Add a WHERE to skip zero-area rows.
SELECT District, the_year, Crop, ProdKg_mod / Areaha_mod AS yield FROM Agriculture WHERE Areaha_mod > 0;
query_yield.sql
4b · Economic productivity — value per hectare (ValueR_mod / Areaha_mod)
Calculate Productivity (defined as Value / surface) in the table Agriculture
Areaha_mod (surface) has to be positive!
Same pattern as 4a but divide ValueR_mod by Areaha_mod.
SELECT District, the_year, Crop, ValueR_mod / Areaha_mod AS ec_prod FROM Agriculture WHERE Areaha_mod > 0;

Hint: Filter first with WHERE, then group and sum.

Show districts with total_production greater than 4000 kg in year 2000.
Hint: HAVING filters after grouping — WHERE runs before.
Add HAVING SUM(ProdKg_mod) > 4000 after GROUP BY. HAVING works on aggregated values; WHERE does not.
SELECT District, SUM(ProdKg_mod) AS total_production FROM Agriculture WHERE the_year = 2000 GROUP BY District HAVING SUM(ProdKg_mod) > 4000;

7a · Full join — all production data with poverty rates
Show all production data together from Agriculture with poverty rates from table poverty rate
Hint: Use the key variable district_id from the table Agriculture and distr_id from the table poverty rate
Use JOIN … ON to specify the key: A.district_id = P.distr_id. Use aliases A and P for readability.
SELECT * FROM Agriculture AS A JOIN poverty_rates AS P ON A.district_id = P.distr_id;
query_join.sql
7b · JOIN + WHERE + GROUP BY — high-poverty districts in 2000
Compute total_production in the year 2000 in districts with poverty > 0.45
Join first, then filter with WHERE on both year and poverty_rate, then GROUP BY district to sum production.
SELECT A.District, SUM(A.ProdKg_mod) AS total_production FROM Agriculture AS A JOIN poverty_rates AS P ON A.district_id = P.distr_id WHERE A.the_year = 2000 AND P.poverty_rate > 0.45 GROUP BY A.District;
