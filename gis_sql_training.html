Filter features with WHERE
Write a query that returns only protected areas of type 'Wetland' with an area larger than 1000 ha. You should get 2 results.
protected_areas
idinteger
nametext

Use WHERE with two conditions joined by AND. Text values need single quotes: type = 'Wetland'
SELECT * FROM protected_areas WHERE type = 'Wetland' AND area_ha > 1000;

04 · SQL
Aggregate with GROUP BY
Count how many features exist per type and calculate the total area per type. Name the columns count and total_area_ha. Order by total area descending.
protected_areas
idinteger
nametext
typetext
area_hainteger
est_yearinteger
regiontext
Use COUNT(*) AS count and SUM(area_ha) AS total_area_ha. Then GROUP BY type.
SELECT type, COUNT(*) AS count, SUM(area_ha) AS total_area_ha FROM protected_areas GROUP BY type ORDER BY total_area_ha DESC;
